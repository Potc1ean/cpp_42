REFERENCE

Permet de definir un autre nom a une variable.

int x = 5;
int& ref = x;

modifier ref modifira x. sauf si :

const int& ref = x;

Si on utilise une ref en parametres d'une fonction cela permet de la modifier
sans la dupliquer.

on peut return une ref pour ne pas avoir a la dup ce qui fait un gros gain de perf.
